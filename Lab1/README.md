# AOIS1: Лабораторная работа по кодам представления чисел

## Цель работы
Реализовать вручную (без `bin`, `int(..., 2)`, `struct`, `numpy`, `decimal`, `eval` и аналогов) представление чисел и арифметические операции:

1. Перевод десятичного числа в:
   - прямой код,
   - обратный код,
   - дополнительный код.
2. Сложение и вычитание в дополнительном коде.
3. Умножение и деление в прямом коде.
4. Операции IEEE-754 binary32 (32 бита): `+`, `-`, `*`, `/`.
5. Кодирование/декодирование и сложение в BCD 2421.

## Ключевые ограничения
- Все ключевые алгоритмы реализованы вручную через битовые массивы.
- Для целочисленных представлений используется фиксированный размер 32 бита там, где это применимо.
- Внутренние преобразования между кодами и арифметика реализованы без готовых конвертеров систем счисления.
- Для IEEE-754 не используются упаковка/распаковка `float` через готовые бинарные API.

## Кратко об алгоритмах
1. `bit_utils.py`
   - Ручные преобразования `int -> bits -> int`.
   - Ручные операции: сложение, вычитание, инверсия, сдвиги, сравнение.
2. `signed_codes.py`
   - Прямой код: бит знака + модуль.
   - Обратный код: для отрицательных инверсия положительного представления.
   - Дополнительный код: инверсия + 1.
3. `twos_complement_arithmetic.py`
   - Сложение двух чисел в доп. коде.
   - Вычитание как `a + (-b)` через битовое отрицание.
   - Контроль переполнения.
4. `sign_magnitude_arithmetic.py`
   - Умножение: сдвиг + сложение частичных произведений.
   - Деление: двоичное длинное деление + ручное формирование 5 десятичных знаков.
5. `ieee754_32.py`
   - Ручное кодирование/декодирование binary32.
   - Операции `+`, `-`, `*`, `/` на уровне знака, экспоненты и мантиссы.
   - Базовые спецслучаи: `NaN`, `Inf`, `0`.
6. `bcd_2421.py`
   - Табличное кодирование/декодирование цифр 0..9 в 2421.
   - Сложение с переносом по десятичным разрядам.
7. `cli/operation_factory.py`
   - Фабрика `OperationFactory` централизует выбор обработчиков для командного и интерактивного режимов.

## Запуск
### Подготовка окружения через uv
```bash
uv sync
```

### CLI-демо (через uv)
```bash
uv run python main.py codes -13
uv run python main.py add 127 -12
uv run python main.py sub 50 73
uv run python main.py mul -7 6
uv run python main.py div 22 7
uv run python main.py ieee add 1.5 2.25
uv run python main.py bcd-add 789 456
```

### Интерактивный ввод пользователя
```bash
uv run python main.py interactive
```

### Запуск тестов
```bash
uv run python -m unittest discover -s tests -v
```

### Отчет покрытия
```bash
uv run python tools/coverage_report.py
```

Скрипт запускает все unit-тесты и печатает покрытие по модулям `src` и итоговый процент.

## Проверяемые сценарии в тестах
- Положительные/отрицательные/нулевые значения.
- Границы диапазонов 32-битных кодов.
- Переполнение для целочисленных операций.
- Корректность деления и дробной части (5 знаков).
- IEEE-754: нормальные числа, субнормалы, `NaN`, `Inf`, деление на ноль.
- BCD 2421: корректные и некорректные кодировки, переносы при сложении.
